1. Верхнее "красное меню" - переключает списки:
- столы (кэш игры)
- Турниры МТТ
- СНГ турниры

2. Левый столбец - это система фильтров в выбранном красном меню:
- Currency - нажимаем, вылезает по центру попап (стандартный а-ля андроидовский, нарисую позже, сделай пока сам "без диза") выбора валюты (Условные деньги, Реальные деньги, Бонусные деньги)
- Game type - попап с выбором типа игры (Холдэм, Омаха, 7Стад...)
- Limit - попап выбора типа лимита (No Limit, Pot Limit)
- Buy-In - попап (Микро 0,01-1, Малый 1,01-9,99, Норма 10,00-... - тут указаны цифрф НИЗШЕГО значения лимитов)
- Seats - попап количество мест за столом (2, 5,6,9,10)
- Speed - скорость игры (Normal, Fast)

!!! по идее все эти данные по каждому столу ты получаешь от сервера и фильтруешь по описанным пунктам, и соответственно строишь список в правом столбце)

3. Левый столбец, поле "Плэй Нау" - информация о выбранном столе (при нажатии на стол в правом столбце - тут показана инфа о нем)

4. Левый столбец (низ) - Время сервера

5. Левый столбец (самый низ) - пункты меню ВСЕГО лобби покера (пока не активны - опишу потом)

6. Правый столбец - список столов / турниров / снг турниров по выбранным фильтрам (п.2)
***к СНГ и МТТ турнирам дам список фильтров завтра

7. Кнопка Refresh обновляет данные в списке по выбранным фильтрам

8. Еще есть текстовое поле внизу правого столбца - там показаны сообщения администрации / реклама (где-то это есть на сервере - надо найтить)
